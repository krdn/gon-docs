SKILL.md 해부도

SKILL.md
--- name: fix-issue description: Fix a GitHub issue argument-hint: [issue-number] disable-model-invocation: true allowed-tools: Bash(gh *), Read context: fork agent: general-purpose model: sonnet --- # Fix GitHub Issue 이슈: !`gh issue view $ARGUMENTS` ## 절차 1. 이슈 $ARGUMENTS 분석 2. 코드 탐색 → 수정 → 테스트
📋
name
/fix-issue로 호출. 소문자+숫자+하이픈, 최대 64자
💬
description
Claude 자동 호출 판단에 사용. 자연어 키워드 포함 권장
🔒
disable-model-invocation
true → 자동 호출 차단. 배포/커밋 등 부작용 있는 스킬에 필수
🔧
allowed-tools
승인 없이 사용 가능한 도구. 와일드카드: Bash(gh *)
🔀
context: fork + agent
격리 서브에이전트에서 실행. Explore/Plan/general-purpose
⚡
$ARGUMENTS & !`cmd`
$ARGUMENTS: 사용자 인자. !`cmd`: 셸 명령 결과 주입 (전처리)

Foreground vs Background

▶ Foreground (기본)

  • • 결과를 기다린 후 다음 진행
  • • 결과가 다음 단계에 필요할 때
  • • 리서치, 탐색 작업에 적합

⏭ Background

  • • 병렬 실행, 완료 시 알림
  • • 독립적인 작업을 동시에
  • • 테스트 실행, 빌드 등에 적합
